## HOW TO STAY SAFE FROM VIRUSES
=================================
General Protection:
1. Install and maintain antivirus software (Windows Defender, Bitdefender, Kaspersky)
2. Keep your operating system updated
3. Don't download files from untrusted websites
4. Avoid opening email attachments from unknown senders
5. Be careful with USB drives from others
6. Use a standard user account, not administrator, for daily tasks
7. Enable firewall protection

Safe Browsing Habits:
- Only visit HTTPS websites (look for padlock icon)
- Don't click on pop-up ads claiming your computer is infected
- Avoid "cracked" software or key generators
- Don't install browser extensions from untrusted sources
- Be wary of "free" movies, music, or games

Email Safety:
- Never open attachments you weren't expecting
- Be suspicious of urgent messages
- Hover over links before clicking
- Don't enable macros in documents
- Verify unexpected requests with the sender by phone

## HOW TO STAY SAFE FROM RANSOMWARE
====================================
Prevention:
1. Maintain offline backups (external drive disconnected after backup)
2. Use cloud backups with versioning (OneDrive, Google Drive, Backblaze)
3. Keep all software updated
4. Use ad-blockers to prevent malicious ads
5. Disable macros in Office applications
6. Use application whitelisting
7. Restrict RDP access (disable if not needed)

Specific Ransomware Protections:
- Enable controlled folder access in Windows
- Use behavior-based antivirus/EDR
- Implement network segmentation
- Use the principle of least privilege
- Disable PowerShell and scripts if not needed
- Block common ransomware file extensions at email gateway

If You Get Infected:
- Immediately disconnect from network
- DO NOT pay the ransom (no guarantee)
- Report to authorities (FBI IC3, CISA)
- Restore from clean backups
- Scan all files before restoring
- Identify how infection occurred

## HOW TO IDENTIFY PHISHING EMAILS
===================================
Red Flags:
1. Urgent or threatening language
2. Generic greetings ("Dear Customer" not your name)
3. Spelling and grammar errors
4. Suspicious sender email address
5. Mismatched links (hover to check)
6. Unexpected attachments
7. Requests for personal information
8. Too good to be true offers

Examples:
- "Your account will be closed in 24 hours"
- "You've won a prize, click here"
- "Verify your password immediately"
- "Invoice attached from unknown company"
- "CEO urgently needs gift cards"

## PASSWORD BEST PRACTICES
===========================
Creating Strong Passwords:
- Use at least 12 characters
- Mix uppercase, lowercase, numbers, symbols
- Avoid dictionary words
- Don't use personal info (birthdays, names)
- Use passphrases (correct-horse-battery-staple)

Password Management:
- Use a password manager (Bitwarden, 1Password, LastPass)
- Never reuse passwords across sites
- Enable two-factor authentication (2FA/MFA)
- Change default passwords immediately
- Don't write passwords on sticky notes

## SAFE INTERNET BROWSING
==========================
Browser Security:
- Keep browser updated
- Use privacy extensions (uBlock Origin, Privacy Badger)
- Clear cookies regularly
- Use private/incognito mode for sensitive sites
- Disable unnecessary plugins (Flash, Java)

Website Safety:
- Check for HTTPS (padlock icon)
- Read URL carefully (amazon.com vs amaz0n.com)
- Don't save passwords in browser
- Log out after banking/shopping
- Use different browsers for different activities

## HOME NETWORK SECURITY
=========================
Wi-Fi Security:
- Use WPA2 or WPA3 encryption
- Change default router password
- Disable WPS (Wi-Fi Protected Setup)
- Hide SSID (optional)
- Enable guest network for visitors
- Keep router firmware updated

Router Settings:
- Change default admin credentials
- Disable remote management
- Enable firewall
- Use DNS filtering (OpenDNS, Cloudflare)
- Regularly check connected devices

## SOCIAL MEDIA SAFETY
=======================
Privacy Settings:
- Review privacy settings regularly
- Limit public posts
- Don't share location in real-time
- Be careful what you post (future employers may see)
- Review tagged photos before approving

What NOT to Share:
- Vacation plans (burglars watch)
- Personal documents (IDs, tickets)
- Financial information
- Work secrets
- Children's full names and schools
- Current location

## MOBILE DEVICE SECURITY
==========================
Phone/Tablet Protection:
- Use strong PIN or biometrics
- Keep OS updated
- Download apps only from official stores
- Review app permissions
- Disable Bluetooth when not in use
- Use Find My Device features
- Encrypt device (enabled by default)

App Safety:
- Read app reviews before installing
- Check permissions (why does a flashlight need contacts?)
- Remove unused apps
- Be careful with QR codes
- Don't jailbreak/root devices

## PUBLIC WI-FI SAFETY
=======================
Risks:
- Man-in-the-middle attacks
- Evil twin access points
- Packet sniffing
- Session hijacking

Protection:
- Use VPN (Virtual Private Network)
- Avoid banking on public Wi-Fi
- Turn off file sharing
- Use HTTPS everywhere
- Forget network after use
- Use your phone hotspot instead

## DATA BACKUP STRATEGY
========================
3-2-1 Rule:
- 3 copies of your data
- 2 different media types
- 1 copy off-site

Implementation:
- External hard drive (weekly backups, then disconnect)
- Cloud backup (Backblaze, iDrive, Carbonite)
- File sync (OneDrive, Google Drive, Dropbox)
- Test restores regularly
- Backup before major updates

## SECURITY CHECKLIST (DAILY/WEEKLY)
=====================================
Daily:
- Check for software updates
- Review bank/credit card transactions
- Empty trash/downloads folder

Weekly:
- Run antivirus scan
- Review recent logins to accounts
- Backup important files
- Clear browser history/cache

Monthly:
- Review account recovery options
- Update passwords for critical accounts
- Check privacy settings on social media
- Review app permissions on phone

## INCIDENT RESPONSE (WHAT TO DO IF HACKED)
============================================
Immediate Steps:
1. Disconnect from internet
2. Change passwords from clean device
3. Enable 2FA/MFA
4. Scan for malware
5. Check for unauthorized transactions
6. Contact banks/credit cards
7. Report identity theft if needed

Account-Specific:
- Email: Check forwarding rules, recovery options
- Social Media: Review authorized apps
- Banking: Monitor transactions, freeze credit
- Work: Inform IT department immediately
